Langage

La programmation est une compétence essentielle pour les métiers de la cybersécurité,
car cela permet d'analyser du code malveillant,
créer ses propes malwares, ses propres reverse-shells...

Une personne travaillant dans ce domaine doit donc consciencieusement
choisir les langages les plus utliles et les plus appropriés pour chaque situation

OS

L' OS d'un ordinateur (ou Système d'exploitation) est un ensemble de programmes permettant
l'accés aux applications d'un appareil, le plus souvent grace à une interface graphique

Pour la bureautique, Windows est l'OS qui domine le marché, mais pour la cybersécurité, les distributions
Linux sont plus intéressantes car ces dernières sont plus modulables que Windows ou Mac OS

Un spécialiste en sécurité informatique doit donc choisir son système d'exploitation
selon les spécificités de ses tâches

Visiter la page sur les système d'exploitation en cliquant ici ou sur OS dans le menu

Faille de sécurité

Une faille de sécurité (ou vulnérabilité) désigne toute faiblesse d'un système qui pourrait permettre
à un pirate informatique de s'y introduire

Les failles de sécurité sont donc au coeur de chaque métier de la cybersécurité

Les personnes exerçant un métier de la "blue-team" (ou défenseurs), doivent trouver des failles dans leurs systèmes,
pour ensuite les réparées et les corrigées, avant qu'un véritable pirate les débusquent

En revanche, les "red teamers" (ou attaquants), doivent, quand à eux, trouver des brèches
dans le système pour essayer de le compromettre,
avertir la blue team pour qu'ils les corrigent,
et ainsi améliorer la sécurité et non-violation de leur sytème
